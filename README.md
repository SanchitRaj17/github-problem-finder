# github-problem-finder

Find difficult bugs, feature requests, or enhancements in GitHub repositories that AI agents cannot easily solve.

## Installation

### Windows (PowerShell)

```powershell
.\scripts\install.ps1 https://github.com/raghavs12/github-problem-finder
```

### Linux/macOS/Git Bash (Windows)

```bash
./scripts/install.sh https://github.com/raghavs12/github-problem-finder
```

**Note for Windows users**: The `chmod` command is not needed on Windows. If you're using Git Bash, you can run the `.sh` script directly. For native PowerShell, use the `.ps1` script instead.
